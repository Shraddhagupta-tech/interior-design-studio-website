Interior Design Studio Website

Overview
This project is a professionally designed interior design studio website focused on luxury residential interiors with a practical, budget-conscious approach.
The website is structured to build trust, clarity, and credibility, while guiding users toward booking a consultation.
Rather than functioning as a generic gallery, the site emphasizes case studies, process transparency, and clear UX flow, reflecting real-world client expectations.

Project Objectives
Establish brand credibility for a residential interior design studio
Present featured work through structured case studies
Clearly communicate key features and design strengths
Drive consultation bookings through focused layout and CTAs

Target Audience
Homeowners seeking professionally designed interiors
Real estate developers evaluating interior design partners

Website Structure
The website includes the following pages:
About – Studio positioning, values, and design philosophy
Features – Key service highlights and differentiators

Case Studies (3 Projects)
Each featured project is presented as an individual case study
Includes design intent, challenges, and visual outcomes
Consultation Form – Dedicated page for booking consultations
This focused structure ensures clarity without overwhelming the user.

Key UX & Design Principles
Minimal, premium visual language with generous white space
Strong hierarchy between emotional headings and functional content
Case-study approach instead of image-heavy galleries
Clear call-to-action placement across pages
Smooth, subtle interactions without unnecessary animation

Responsiveness
Mobile-first layout considerations
Vertical storytelling on mobile devices
Balanced spacing on tablet views
Immersive visuals and grid alignment on desktop
Built using Bootstrap’s grid system to ensure consistent behavior across breakpoints.

Tech Stack
HTML5
CSS3
Bootstrap

What This Project Demonstrates
Real-world frontend layout planning
UX-driven design decisions aligned with business goals
Ability to present design work professionally
Clean, maintainable structure using a responsive framework
